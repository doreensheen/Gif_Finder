Gif Finder
by Doreen Sheen

Click on a button to display 10 gifs from GIPHY API related to the text on the button clicked.

Click "Show More" at the bottom of the page to display 10 more gifs from GIPHY API.

"Add an emotion" will create a new button with the text entered. That button can then be clicked to display 
10 gifs from the GIPHY API related to the text on the button.

Click "Save" on any of the gifs you'd like to save as a favorite. To see your favorites, click "Saved Favorites"
at the top of the page.


Note:

favorites.html is disabled, a favorites modal was created instead in the index.html page
